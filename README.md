# Advanced Python Programming: From Basics to Mastery

This repository contains all the code samples, practice projects, and advanced examples featured in the **Advanced Python Programming** course.

## Repository Structure

- **`/basics`**: Core Python syntax, loops, functions, and exception handling.
- **`/advanced`**: Object-Oriented Programming (OOP), REST APIs (Flask), Deep Learning (PyTorch), and CI/CD configurations.
- **`/testing`**: Unit testing examples using `pytest`.
- **`/projects`**: Practical practice projects including a Number Guessing Game and a Task Manager.

## How to Use

1. **Clone the repository**:
   ```bash
   git clone https://github.com/manus-ai/python-course-samples.git
   ```
2. **Install dependencies** (for advanced samples):
   ```bash
   pip install flask torch pytest
   ```
3. **Run the samples**:
   Navigate to any directory and run the Python files:
   ```bash
   python basics/if_else_grade.py
   ```

## Course Topics Covered

- Python 3.14 Evolution (JIT, Free-threading)
- Core Syntax & Data Types
- Control Flow & Exception Handling
- Functional & Object-Oriented Programming
- Unit Testing & Code Coverage
- REST API Development with Flask
- Deep Learning with PyTorch
